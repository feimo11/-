# 孕期记录应用功能说明

## 应用介绍

这是一个专为孕妇设计的孕期记录应用，通过直观的日历界面和详细的时间线，帮助准妈妈轻松追踪孕期进展。应用采用响应式设计，支持在手机、平板和电脑上使用，数据本地存储确保隐私安全。

## 核心功能

### 1. 日历功能

- **完整月历**：支持左右翻月导航
- **农历显示**：每个日期显示对应的农历日期
- **特殊标记**：
  - 今天：浅粉色背景
  - 怀孕起始日：渐变粉色背景
  - 孕期范围内：浅色背景
  - 有笔记的日期：底部粉色小圆点

### 2. 怀孕起始日期设置

- 在日历中选择日期，点击「确认选择此日期为怀孕起始日」
- 系统自动计算孕周、孕天和预产期
- 数据通过 localStorage 持久化保存
- 支持修改和清除起始日期

### 3. 孕期状态追踪

- **环形进度条**：直观显示孕周进度
- **详细信息**：显示当前孕周+天（如 8周+3天）
- **三孕期标记**：自动识别孕早期、孕中期、孕晚期
- **预产期计算**：基于起始日期+280天自动计算

### 4. 笔记功能

- **快速添加**：在笔记页面直接为今天添加笔记
- **日期详情**：点击日历日期弹出详情，可添加/删除笔记
- **笔记标记**：有笔记的日期在日历上显示粉色小圆点
- **笔记列表**：按日期倒序展示所有笔记，包含详细信息

### 5. 孕期时间线

- **重要里程碑**：11个关键节点（受孕开始→预产期）
- **当前阶段**：高亮显示当前所处孕期阶段
- **日期标记**：每个里程碑显示对应日期

### 6. 孕期小贴士

- 根据当前孕期阶段（早/中/晚期）自动推送相关建议
- 提供健康、饮食、生活等方面的实用信息

## 界面布局

### 底部导航

- 🏠 首页：显示孕期状态和今日信息
- 📅 日历：选择起始日期和查看日期详情
- 📝 时间线：查看孕期里程碑
- 📔 笔记：管理所有笔记

### 数据展示

- **首页**：顶部显示今日日期、星期、农历；中部显示孕期状态和进度；底部显示今日详细信息
- **日历**：月视图日历，支持日期选择和笔记管理
- **时间线**：按时间顺序展示孕期里程碑
- **笔记**：按日期倒序展示所有笔记，支持快速添加

## 技术特性

### 前端技术

- **纯HTML5+CSS3+JavaScript**：无框架依赖，加载速度快
- **响应式设计**：适配各种屏幕尺寸
- **本地存储**：使用 localStorage 存储数据
- **农历计算**：内置农历算法，无需网络

### 数据安全

- **本地存储**：所有数据存储在浏览器本地，不上传到服务器
- **隐私保护**：敏感信息仅保存在用户设备上
- **数据持久**：刷新页面或重新打开应用数据不丢失

### 性能优化

- **轻量级**：单文件应用，无外部依赖
- **快速加载**：优化代码结构，减少不必要的计算
- **流畅动画**：使用 CSS 过渡效果，提升用户体验

## 使用方法

### 首次使用

1. 打开应用后，在日历页面选择怀孕起始日期
2. 点击「确认选择此日期为怀孕起始日」
3. 系统自动计算并显示孕期信息

### 记录笔记

1. **方法一**：在笔记页面顶部输入框直接添加今天的笔记
2. **方法二**：在日历页面点击任意日期，在弹出的详情中添加笔记
3. 每条笔记会自动记录添加时间

### 查看孕期信息

- 首页显示当前孕周、孕天和孕期阶段
- 时间线页面查看重要里程碑
- 日历页面查看特定日期的孕期信息

## 部署指南

### 本地访问

1. 保存 `pregnancy_tracker.html` 文件到本地
2. 启动本地服务器（如 `python -m http.server 8080`）
3. 浏览器访问 `http://localhost:8080/pregnancy_tracker.html`

### 网络访问

1. 将文件部署到静态网站托管服务（如 Vercel、Netlify、GitHub Pages）
2. 通过生成的 URL 访问应用
3. 在手机浏览器中打开 URL，可添加到主屏幕

## 常见问题

### 数据丢失怎么办？

- 应用使用 localStorage 存储数据，浏览器清除缓存可能导致数据丢失
- 建议定期截图保存重要信息
- 可考虑手动导出数据（通过浏览器开发者工具）

### 如何在多设备间同步数据？

- 由于数据存储在本地，多设备间无法自动同步
- 可通过以下方式手动同步：
  1. 在原设备上导出 localStorage 数据
  2. 在新设备上导入数据

### 预产期计算准确吗？

- 应用使用标准的 280 天计算方法
- 实际预产期可能因个人情况有所差异
- 最终预产期以医生诊断为准
